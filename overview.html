Tools and Process

  • XCode

The process of creating the app consists of:

  1. Watching XCode tutorials on Youtube
  2. Learning Swift Language
  3. Coding all the thngs on the app
  4. Exporting it to a Phone

What We Learned

Through this project, we learned:

  • How to use XCode to create a IOS app
  • How to use the Programming language : Swift
  • How to design a app's interface
  • How to apply previously learned computer science concepts to app development

How It Works

The visual novel game is created by coding different scenes using Python and Pygame. The scenes are then added together to create a cohesive storyline. The game is controlled by the player's choices, which determine the outcome of the story.

What I Created

I followed a tutorial for a tick tac toe app, but I modified it with sound effects that there is a "Ding" when a player wins and a "Buzz" when a invalid move is made.

Connection to Previous Learning

The project builds upon previously learned computer science concepts such as programming logic, data types, and control structures. Additionally, the project incorporates design principles such as user experience and visual design.

The project also has connections to other subjects such as science, specifically in the areas of physics and biology. For example, the game could incorporate scientific concepts such as genetics or physics principles to enhance the storyline and gameplay.

Terminology, Tools, and Processes

The project utilizes several programming and design concepts and tools, including:

  • Swift programming language
  • Integrated Development Environment (IDE), such as PyCharm
  • User Interface (UI) design principles
  • Version control tools, such as Git and GitHub

Throughout the project, we also learned new terminology and processes specific to app development, such as auto layout and Model-View-Controller (MVC).

Outcomes Compared to Goals

The outcomes of the project met the goals originally set, which were to create a app using XCode, and learn to export the app so it could be played on a phone. Additionally, the project allowed us to apply previously learned computer science concepts to a new context, and to develop our skills in programming and design.
