How It Works

Here's the code for a simple Tip Calculator app that allows the user to enter a bill amount and a tip percentage, and then calculates the tip amount and the total bill amount:

  
    import UIKit

    class ViewController: UIViewController {

        @IBOutlet weak var billAmountTxt: UITextField!
        @IBOutlet weak var tipPercentTxt: UITextField!
        @IBOutlet weak var calculateTipBtn: UIButton!

        @IBOutlet weak var tipAmountLbl: UILabel!
        @IBOutlet weak var billTotalLbl: UILabel!

        @IBAction func calculateTip(_ sender: Any) {
            
            // Convert bill amount text to a Double
            let billAmount:Double = Double(billAmountTxt.text!)!
            
            // Calculate tip amount based on bill amount and tip percentage
            let tipAmount:Double = (Double(tipPercentTxt.text!)! / 100) * billAmount
            tipAmountLbl.text = String(format: "$%.02f", tipAmount)
            
            // Calculate total bill amount and update label
            billTotalLbl.text = String(format: "$%.02f", billAmount + tipAmount)
            
        }
        
        override func viewDidLoad() {
            super.viewDidLoad()
            
        }

    }

Here's an explanation of what each part of the code does:

  • import UIKit: This line imports the UIKit framework, which contains classes for building iOS applications.
  • class ViewController: UIViewController: This declares a new class called ViewController, which inherits from UIViewController, one of the built-in view controller classes in UIKit. This means that ViewController is responsible for managing the user interface of the app.
  • @IBOutlet weak var: These lines declare six different user interface elements as outlets, which means that they can be accessed and modified programmatically. The billAmountTxt and tipPercentTxt are two text fields that allow the user to input the bill amount and the tip percentage, respectively. calculateTipBtn is a button that the user taps to calculate the tip. tipAmountLbl and billTotalLbl are two labels that display the tip amount and the total bill amount, respectively.
  • @IBAction func calculateTip(_ sender: Any): This line declares an action method called calculateTip, which is triggered when the user taps the calculateTipBtn button.
  • let billAmount:Double = Double(billAmountTxt.text!)!: This line converts the bill amount entered by the user from a string to a double, and assigns it to a constant called billAmount.
  • let tipAmount:Double = (Double(tipPercentTxt.text!)! / 100) * billAmount: This line calculates the tip amount based on the bill amount and the tip percentage entered by the user. The tip percentage is divided by 100 to convert it from a percentage to a decimal, and then multiplied by the bill amount to get the tip amount. The result is assigned to a constant called tipAmount.
  • tipAmountLbl.text = String(format: "$%.02f", tipAmount): This line formats the tipAmount as a currency string with two decimal places using the String(format:) method, and assigns it to the text property of the tipAmountLbl label.
  • billTotalLbl.text = String(format: "$%.02f", billAmount + tipAmount): This line calculates the total bill amount by adding the bill amount and the tip amount, formats it as a currency string with two decimal places using the String(format:) method, and assigns it to the text property of the billTotalLbl label.
  • override func viewDidLoad(): This method is called when the view controller is loaded into memory. In this case, it doesn't do anything, so it's empty.
